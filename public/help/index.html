peer puzzle
Peer Puzzle is an experiment in realtime collaboration using mobile devices.

To play, find some friends to join along, or just open the game in multiple browser windows.

Each person should select 'join or start a game' and type the same room name.

Once you start, you'll notice holes in the wall of your maze. Send the marble into one to transfer it to someone else.

Work together to get to the green square and win!
